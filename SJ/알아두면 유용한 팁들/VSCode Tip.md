# VSCode Tip

## VSCode에 Vim모드 실행

나는 intellij에서 항상 vim모드로 사용했기 때문에 vim모드가 더 익숙하여 VSCode에서도 적용해보려고 찾아 보았다.

1. VSCode Extension 실행
    - cmd + shift + x
2. vim 검색 후 설치
    - 하단에 -- normal -- 이라고 뜸.
3. Command 모드에서 Toggle Vim mode 명령어 실행
    - cmd + shift + p
    - vim mode 검색, Toggle Vim mode 실행
    - -- VIM: DISABLED --
